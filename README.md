# AltaVista Precios

B2B Optics Pricing Platform

## Tech Stack
- Next.js 15.5.4
- TypeScript
- Supabase
- Tailwind CSS v4
- shadcn/ui

## Setup

1. Install dependencies:
```bash
pnpm install
```

2. Set up environment variables:
Create `.env.local` with your Supabase credentials.

3. Run development server:
```bash
pnpm dev
```

## Build
```bash
pnpm build
```

## Deployment
Deployed on Vercel
